背景

你有没有遇到过这种情况？

我只想查看一个文件的第一行，但是它太大了，打开它太慢了。

这时候就是本程序大展身手的好时机了。

使用方法

  • [-L] - 是否显示行号（可选）
  • [-N num] - 要显示的行数（默认999999）（可选）
  • path - 文件路径（必须）

把编译过的代码放到某个path路径里就可以用cmd玩了。
