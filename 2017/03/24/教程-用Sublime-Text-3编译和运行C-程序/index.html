前言

咳咳，如果你dev用惯了，看他不爽，想换个IDE，那么推荐用Sublime Text这款软件。由于服务器在国外，所以下载可能会有点慢，所以提供一个百度网盘链接：jvgp。链接挂了跟我说。

添加Build System
