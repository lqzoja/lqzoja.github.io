题面

给定一棵n个点的带权树，求树上最长的异或和路径。

Input

4
1 2 3
2 3 4
2 4 6

Output

7

题解

可以注意到 a -> b 的异或和 = a -> root 的异或和 ^ b -> root 的异或和。所以用 Trie 保存所有点到根节点的异或和，再枚举每一个异或和，在Trie上尽量找与他每一位都相反的就行了（如果存在）。
